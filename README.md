## GsMcStatus

为 GsCore 提供的 Minecraft 服务器状态查询插件。

### 使用方法

- `mc <地址或别名>`: 查询服务器状态
- `mc add <别名> <地址>`: 添加服务器别名，例如 `mc add test mc.example.com:25565`
- `mc del <别名>`: 删除别名
- `mc help`: 获取帮助信息

### 安装
将此文件夹放置在 `gsuid_core/plugins` 目录下。`git clone` 或者传 `.zip` 解压均可。
